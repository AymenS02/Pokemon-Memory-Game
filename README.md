# Pokémon Memory Game

Welcome to the Pokémon Memory Game! Test your memory and Pokémon knowledge by clicking on unique Pokémon images. The goal is to click on each Pokémon only once, and your score increases with each successful click. If you click on the same Pokémon twice, your score resets, but don't worry – you can always try to beat your best score!

## Getting Started

1. Clone the repository to your local machine:

git clone https://github.com/AymenS02/pokemon-memory-game.git

2. Navigate to the project directory:

cd pokemon-memory-game

3. Install dependencies:

npm install

4. Start the development server:

npm start

5. Open your browser and go to `http://localhost:3000` to play the game.

## How to Play

- Click on Pokémon cards to get a point
- The order of Pokémon cards shuffles after each click.
- Avoid clicking on the same Pokémon twice to maintain your score.
- Try to achieve the highest score possible!

## Features

- Utilizes React for a dynamic and interactive user interface.
- Fetches Pokémon data from the [PokéAPI](https://pokeapi.co/) to provide a diverse set of Pokémon for the game.
- Randomizes Pokémon card order for added challenge.
- Keeps track of your current score and best score.
